# To-Do List PySide6 📝

Una aplicación de escritorio moderna y minimalista para gestionar tareas pendientes, construida con **Python** y **PySide6** (Qt for Python). La interfaz cuenta con un diseño oscuro (Dark Mode) basado en la paleta de colores *Catppuccin*.

## ✨ Características

- **Añadir tareas:** Crea nuevas tareas rápidamente presionando Enter o haciendo clic en el botón.
- **Marcar completadas:** Sistema de checkboxes con efecto visual de tachado y atenuado de texto.
- **Limpieza masiva:** Elimina todas las tareas completadas con un solo clic.
- **Diseño Moderno:** Interfaz responsiva con bordes redondeados, animaciones hover y paleta de colores profesional.
- **Arquitectura Limpia:** Separación de responsabilidades entre Interfaz (UI), Lógica y Ejecución.
